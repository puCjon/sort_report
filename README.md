# 氣泡排序法
#### 氣泡排序是一種較基本的排序演算法。它重複地訪問要排序的數值，一次比較兩個數字，如果他們的順序錯誤就把他們的位子互相交換。一直重複執行直到不需要在交換時，也就代表數列已經完成排序。這個演算法的名字由來是因為越小的元素會經由交換慢慢浮到數列的頂端，就如同碳酸飲料中的氣泡最終會上浮到頂端一樣，所以才得到氣泡排序法這個名字。
# 選擇排序法
#### 選擇排序法的工作原理是每一次從待排序的數據數值中選出最小的一個元素，存放在序列的最前面的位置，直到全部待排序的數據排完。選擇排序是不穩定的排序方法，因為在交換過程中可能改變相等元素的相對順序。
# 插入排序法
#### 插入排序法將數組分為已排序和未排序兩部分，剛開時的排序部分只有第一個元素。然後從未排序部分取出第一個元素，插入到已排序部分的正確位置，重複此過程直到沒有了沒排序數值。插入排序在實務上通常比其他簡單的排序演算法更有效率。
# 合併排序法
#### 合併排序是一種分治演算法。它將數列分成兩半，遞迴地對兩半進行排序，然後將兩個已排序的子數組合併成一個有序數組。這種演算法的時間複雜度為O(n log n)，比簡單排序演算法更有效率。合併排序是穩定排序，且在實務應用中廣泛使用。

# 綜合比較
### 基礎類 (O(n^2)$)：
#### 氣泡排序：相鄰比較並交換，效率最低。
#### 選擇排序：反覆選出最小值放最左邊。優點是交換次數少，但不穩定。
#### 插入排序：像排撲克牌，將新牌插進已排好的序列。在資料近乎排序時速度最快。
### 進階類 (O(nlogn))：
#### 合併排序：採用分治法，先拆解再合併。效能最穩定且具穩定性，但需要較多記憶體空間。
### 總結：小數據或近乎排序使用插入排序法；大數據且空間足夠則用合併排序法。
